# 🏥 Hospital Analytics Dashboard | Power BI

## 📌 Project Overview
The Hospital Analytics Dashboard is an end-to-end healthcare analytics project built using Power BI to analyze patient admissions, diagnoses, bed occupancy, billing, insurance coverage, doctor performance, and patient feedback.

## 🎯 Business Problem
Hospitals generate large volumes of operational and patient data. Without analytics, it is difficult to track admissions, monitor bed utilization, evaluate doctor workload, analyze billing trends, and improve patient satisfaction. This dashboard provides actionable insights for better decision-making.

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel / CSV
- Data Modeling
- Data Visualization

## 📊 Key KPIs
- Total Patients
- Total Admissions
- Total Billing Amount
- Total Insurance Coverage
- Average Feedback Rating
- Bed Occupancy Rate
- Diagnosis Distribution
- Doctor-wise Patient Count

## 🔍 Key Insights
- Analyze patient admission trends.
- Monitor bed occupancy across categories.
- Evaluate doctor workload and performance.
- Compare billing amounts with insurance coverage.
- Track patient satisfaction using feedback ratings.
- Identify common diagnoses and test utilization patterns.

## 💡 Business Recommendations
- Optimize bed allocation and resource planning.
- Improve patient care using feedback insights.
- Monitor high-frequency diagnoses for preventive healthcare programs.
- Enhance insurance claim processing efficiency.
- Balance doctor workload for better operational efficiency.

## 🚀 Future Enhancements
- Predictive patient admission forecasting.
- Real-time hospital monitoring dashboards.
- Disease outbreak trend analysis.
- Patient risk prediction using machine learning.
